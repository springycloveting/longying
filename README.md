## 运行方式

### Web 版本 (推荐 - 适合 Windows 用户)

**方式一：直接运行**
1. 双击 `run_web_simple.bat`
2. 浏览器会自动打开 http://localhost:5000

**方式二：虚拟环境运行**
1. 双击 `run_web.bat` 首次会自动创建虚拟环境并安装依赖
2. 浏览器会自动打开

**打包成 EXE**
1. 双击 `build_web.bat` 打包
2. 生成的 EXE 在 `dist/龙胤立志传Web修改器.exe`

### GUI 版本 (PyQt6)

**Windows 用户**
1. 双击 `run.bat` 首次会自动安装依赖并启动

**打包成 EXE**
- 双击 `build_exe.bat` 打包
- 生成的 EXE 在 `dist/龙胤立志传存档修改器.exe`

### Linux/Mac 用户

```bash
# GUI版本
chmod +x run.sh
./run.sh

# Web版本
python3 web_server.py
# 然后浏览器打开 http://localhost:5000

# 打包
./build_exe.sh
```

## GUI 功能说明

### 界面布局
- **左侧**: 角色列表，支持搜索筛选
- **右侧**: Tab 页签切换不同功能

### 基本属性页
- 快捷操作: 一键全满、治愈、复活、金钱满、声望满
- 状态栏: 生命/体力/内力（当前值/最大值）
- 基础属性: 力道/灵巧/智力/意志/体质/经脉
- 武学属性: 内功/轻功/绝技/拳掌/剑法/刀法/长兵/奇门/射术
- 技能属性: 医术/毒术/学识/口才/采伐/木植/锻造/炼丹/烹饪
- 其他信息: 声望/恶名/金钱

### 武功技能页
- 武功列表: 显示已学武功详情
- 添加武功: 搜索并添加新武功
- 编辑武功: 修改等级和伤害加成
- 删除武功: 移除已学武功（自动更新装备索引）
- 全部满级: 一键所有武功满级

### 物品装备页
- 装备信息: 武器/护甲/头盔/鞋子/饰品
- 物品列表: 背包内所有物品

### 人际关系页
- 好友列表
- 仇人列表
- 徒弟列表

### 工具页
- 批量操作: 所有角色属性满、武功满级、治愈、复活
- 存档信息: 显示角色总数等

## Python API 使用示例

```python
from full_editor import *

# 加载存档
data = load_data()
_, hero = find_hero(data, '杨春')

# 修改属性
modify_base_attr(hero, 0, 999)      # 力道改为999
modify_fight_skill(hero, 4, 999)    # 剑法改为999
modify_living_skill(hero, 2, 100)   # 学识改为100

# 修改状态
modify_status(hero, hp=9999, power=9999, mana=9999)

# 修改声望金钱
modify_fame(hero, fame=99999)
modify_money(hero, 999999)

# 添加武功
add_skill(hero, 100, 10)  # 添加ID为100的武功，等级10

# 删除武功（自动更新装备索引）
remove_skill(hero, 0)  # 删除第一个武功

# 所有武功满级
all_skills_max(hero)

# 一键全满
max_all_attrs(hero, 999)

# 治愈/复活
heal_hero(hero)
revive_hero(hero)

# 保存
backup_file('Hero')
save_data(data, 'Hero')
```

## 注意事项

1. **修改前备份** - 保存时会自动备份
2. **存档位置** - Hero 文件是主存档，修改后需要放回原位
3. **属性值范围** - 建议不要设置过大的数值，可能导致游戏异常
4. **武功ID** - 可在 skill_names.json 中查看所有武功
5. **属性ID** - 可在 spe_attr_map.json 中查看所有属性
6. **删除武功** - 已正确处理装备索引更新，不会影响已装备武功
